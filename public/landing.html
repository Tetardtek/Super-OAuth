Integration

3 etapes. C'est tout.

1. Creez un tenant

Un client_id + client_secret. Configurez vos providers (Discord, Google, GitHub, Twitch) et vos redirect URIs.

2. Redirigez vos users

GET /oauth/authorize avec votre client_id et un code_challenge PKCE. On gere le reste.

3. Echangez le code

POST /oauth/token avec le code + code_verifier. Vous recevez un access_token signe avec le secret de votre tenant.
